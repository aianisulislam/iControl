# Code of Conduct

## The short version

Be straightforward and respectful. Disagreements about code are fine; personal attacks are not.

## Expected behaviour

- Criticise ideas, not people.
- Give and receive feedback without taking it personally.
- Stay on topic — this is a technical project, not a social platform.

## Unacceptable behaviour

- Harassment, discrimination, or personal attacks of any kind.
- Sustained off-topic disruption.
- Publishing others' private information without consent.

## Scope

This applies to GitHub issues, pull requests, and any other project spaces.

## Enforcement

Violations can be reported by opening a private issue or contacting the maintainer directly via GitHub. Reports will be reviewed and acted on at the maintainer's discretion, up to and including a permanent ban from the project.
